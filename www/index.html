About TEAMಎISE

TEAMwISE is an abbreviation of ‘Tool for the Exploration and Analysis of Movement data Within Immersive Synchronised Environments’. We aim to build a tool that is an open source, web-based stand-alone application. The current implementation is built based on the Cesium platform which is an open-source JavaScript library for 3D globes and maps.

Currently, TEAMwISE supports KML formal, to know more about the tool, please go to our user guide!

Features

Movement Data

Visualisation

TEAMwISE can stream movement data on terrain or imagery, so you can observe your collected data with realistic detial.

3D Tiling

Visualisation

3D Tiles can be clamped to terrain, and stream of geospatial data as the supply. Points, polylines, and polygons can be added on data visualisation.

Multiple Device Display

Syncronisation

TEAMwISE allows multiple researchers to work on the same data set and dynamically work together, or to visualise different aspects of this data set.

Stereoscopic 3D

Visualisation

TEAMwISE can provide stereoscopic 3D visualisation on large a display or a mobile device.
